# Piper

## What is Piper
Piper is a mini modular tracker in the spirit of BuzzTracker, Max/MSP, SunVox, PD, and a bunch of other niche software, with a very simple, portable interface, and every synth & effect built as a lua plugin. The idea is it runs anywhere, is open, and anybody can easily build for it.

## Installation
Download and install [Love2d](https://love2d.org/) from the website or using any major package manager e.g.

`sudo apt-get install love` / `brew install love` / etc

## Running
Drag the folder to a running love2d instance, or else just:

```
cd piper
love .
```
